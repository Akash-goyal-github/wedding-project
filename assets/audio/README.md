# Wedding Background Music

Drop your wedding background music file here named exactly:

```
wedding-bg.mp3
```

### Tips
- Recommended: a soft shehnai / classical instrumental, ~3-5 min, loops well
- File size: keep under 5 MB for fast mobile loading
- The music auto-starts (fade-in) after the gate entrance closes
- The floating 🎵 button (bottom-right) lets guests toggle it on/off
- Their preference is remembered across page visits via localStorage
